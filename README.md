# learn-go-with-tests

learn-go-with-tests is my learning repository that follows the free [Learn Go With Tests][quii] online resource.

## Installation

You'll need [Go](https://go.dev) and [Task](https://taskfile.dev/#/installation) on your system.

## Usage

```bash
task test
```
